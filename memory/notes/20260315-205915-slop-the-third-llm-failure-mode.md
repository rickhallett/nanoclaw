LLMs have a third failure mode beyond hallucination and refusal: slop. Output that is syntactically valid, passes type checks, passes tests, and is wrong in ways that take a skilled human time to identify. Confident, coherent, contextually plausible, and misleading because it optimises for the shape of good work rather than the substance. Not a bug to fix but the nature of probabilistic generation. The response is to build verification density, not demand determinism.
