Six failure modes identified in pilot study: (1) spinning to infinity - recursive self-reflection without decisions, fix: 'decision or analysis?', (2) sycophantic amplification loop - human proposes, agent validates, neither brakes, fix: bearing check against true north, (3) cognitive deskilling - skill atrophy through delegation, manifests across sessions not within, METR RCT 2025 found 40-point perception-reality gap, (4) cold context pressure, (5) hot context pressure, (6) compaction loss - decisions not on file before context window death.
