Bottom-up data flow, top-down control: L0 Weights (frozen), L1 Tokenisation (budget finite), L2 Attention (O(n^2), not observable), L3 Context Window (saturation, lost-in-middle, compaction is discontinuous), L4 Generation (autoregressive, no lookahead), L5 API (only calibrated layer), L6 Harness (orchestration modes), L7 Tools (empirical contact with reality), L8 Agent Role (system prompt, high-attention), L9 Thread Position (self-reinforcing anchoring), L10 Multi-Agent (precision increases, accuracy does not), L11 Cross-Model (independent signal from different priors), L12 Human (irreducible, not scalable, catches oracle-level errors).
