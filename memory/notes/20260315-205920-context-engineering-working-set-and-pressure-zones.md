Novel concepts with no clean literature equivalent. Working set (Denning 1968 isomorphism): minimum context for correct output. Cold context pressure: too little on-file context pushes agent to pattern-match. Hot context pressure: too much in-thread raises compaction risk. Dumb zone: operating outside effective context range, syntactically valid but semantically disconnected. Compaction loss: context window death with decisions not on file = permanent loss, binary and total. Context quality loop: clean code produces better context which produces cleaner code; slop compounds.
