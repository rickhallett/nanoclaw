Not using TDD/automated tests when working with coding agents, even though tests are now effectively free. The asymmetry: cost of not testing is unchanged; cost of testing has dropped to near zero. Willison: tests are non-optional when using coding agents. A specific variant of Deferred Reckoning distinguished by the economic argument.
