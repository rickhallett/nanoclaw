# Weeks 1–2: subprocess + collections
## Interactive Python curriculum with IPython checkpoints

### How to run

```bash
cd /Users/mrkai/code/halo
uv run python data/advisors/guido/curriculum/scripts/day01_defaultdict_basics.py
```

At each checkpoint, you drop into an IPython REPL with all variables loaded.
Type the exercises out. Predict output before running. Ctrl-D to continue.
