Using the open Google Chrome browser and the steer CLI for all GUI interaction:

1. Navigate the eBay tab to https://www.ebay.co.uk/sh/lst/ended
2. Use steer ocr to find the PS5 listing in the ended/unsold items. Filter with grep for PS5 or PlayStation.
3. Click the Relist button/link for that listing
4. Review the relisted draft — check title, price, description, photos, item specifics, postage, and condition all carried over
5. If anything looks wrong or missing, note it in the job YAML updates
6. Do NOT submit the listing — leave it ready for review

Remember: alias steer once at the start, observe selectively with grep, act decisively.
Update the job YAML with progress after finding the listing and after reviewing the relist draft.
When done, write your summary, clean up, then type /exit to end the session.
